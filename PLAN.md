# 📚 AI 스터디 어시스턴트 개발 계획서

본 프로젝트는 학생들이 PDF 수업 자료를 업로드하면 AI가 핵심 내용을 요약하고 퀴즈를 생성하여 학습 효율을 높여주는 웹 서비스입니다.

## 🛠 1. 기술 스택

- **Frontend:** HTML5, CSS3 (Vanilla CSS), JavaScript (ES6+)
- **PDF 처리:** [pdf.js](https://mozilla.github.io/pdf.js/) (브라우저에서 PDF 텍스트 추출)
- **AI API:** Google Gemini API (요약 및 퀴즈 생성)

## 📝 2. 주요 기능

1.  **PDF 업로드 및 텍스트 추출:** 사용자가 PDF 파일을 선택하면 내용을 읽어 텍스트로 변환합니다.
2.  **핵심 내용 요약:** 추출된 텍스트를 AI에게 전달하여 구조화된 요약본을 생성합니다.
3.  **학습 퀴즈 생성:** 본문 내용을 바탕으로 객관식/주관식 퀴즈를 생성하고 정답 확인 기능을 제공합니다.
4.  **반응형 UI:** 모바일과 데스크탑에서 모두 사용 가능한 깔끔한 디자인을 제공합니다.
